# MeetingSecretary（会议整理）

将指定数量的群聊消息整理为 QQ 合并转发（合订本）发送到当前会话，并可屏蔽指定用户的发言，避免无关内容污染整理结果。

## 功能

- 指令 `/meeting <消息数量:int> [屏蔽用户ID ...]`
- 以指令前一条消息为起点，逆向回溯指定数量的消息
- 合并转发每次最多 100 条；超过 100 自动拆分多次发送
- 支持屏蔽多个用户：命令中可传多个 ID（也支持用英文逗号分隔）

## 用法示例

- `/meeting 80`
- `/meeting 200 123456 987654`
- `/meeting 50 123456,987654`

## 平台支持

当前主要针对 OneBot v11（`aiocqhttp`）使用的 QQ 个人号协议端（如 Napcat/Lagrange 等）。插件会优先尝试从协议端拉取群历史；若拉取失败则回退使用插件启用后记录的本地缓存。
